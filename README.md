# Tshwane Leadership Foundation Website

## Overview

The Tshwane Leadership Foundation Website is a full-stack web application developed to support the digital presence of the Tshwane Leadership Foundation. The platform provides an accessible and user-friendly experience, allowing visitors to learn about the organisation, create an account, log in securely, and submit donations online.

This project demonstrates practical experience in full-stack web development, user authentication, database integration, and responsive web design.

---

## Features

* Responsive and user-friendly interface
* User registration and account creation
* Secure user login functionality
* Online donation submission
* Payment page
* Database integration using MySQL
* Dynamic content powered by PHP
